# 📁 Project Structure / 项目目录结构

**English** | **中文**

---

## English

```
stock-monitor/
├── .github/workflows/         # GitHub Actions CI/CD
├── .gitignore                 # Git ignore rules
├── config.example.json        # Example configuration
├── DIRECTORY_STRUCTURE.md     # This file
├── LICENSE                    # MIT License
├── README.md                  # Documentation
├── requirements.txt           # Python dependencies
├── setup.py                   # Setup wizard
├── start.sh                   # Start script (Linux/Mac)
├── stock_monitor.py           # Main program
└── test_install.py            # Installation test
```

### File Descriptions

| File | Description | Required |
|------|-------------|----------|
| `stock_monitor.py` | Main program with monitoring & push | ✅ |
| `config.example.json` | Example configuration template | ✅ |
| `config.json` | User configuration (generated by setup.py) | ✅ |
| `setup.py` | Interactive setup wizard | ✅ |
| `test_install.py` | Installation verification script | Optional |
| `start.sh` | Startup script for Linux/Mac servers | Optional |

### First-time User Workflow

1. **Download** / 下载: `git clone`
2. **Install** / 安装: `pip3 install -r requirements.txt`
3. **Configure** / 配置: `python3 setup.py`
4. **Test** / 测试: `python3 test_install.py`
5. **Run** / 运行: `python3 stock_monitor.py`

### 文件说明

| 文件 | 说明 | 必需 |
